centOS上因为看php源码中没有json模块，于是采用pecl自动编译安装：# yum install php-devel# yum install php-pear# yum install gcc# pear install pecl/json
 
# cd /etc/php.d# echo “extension=json.so” &gt; json.ini# service httpd restart
