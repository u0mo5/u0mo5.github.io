在使用Laravel框架前，要先做一些准备工作。这也是我这几天折腾出来的心得。

1️⃣安装Composer

这个非常简单，通过PhpStorm很容易办到。
打开PhpStorm的设置⌘+,找到Languages&Frameworks->PHP目录下的Composer，先配置好PHP所在目录，然后点击Click here to download from get composer.org，PhpStorm就会自动为你安装Composer。


2️⃣安装Node.js

打开https://nodejs.org/，下载node.js，然后直接运行安装即可。
接下来打开PhpStorm配置Node.js。打开PhpStorm的设置⌘+,找到Languages&Frameworks目录下的Node.js and NPM，配置一下Node.js和NPM包的所在位置。


3️⃣使用PhpStorm创建Laravel项目。


创建新项目

选择Composer Project，在Filter packages中输入laravel/laravel，在Available packages中选中，然后点击Create创建项目。

创建完成后，点击Gulp，系统会提示你"Failed to list tasks"。运行Gulp会得到如下结果：

发现问题，那就解决问题喽。
先在终端Terminal输入node -v检查一下Node.js的版本，然后再输入sudo npm install --global gulp安装Gulp，再输入npm install。
安装完成后，运行一下Gulp发现已经可以运行了。
