利用时间戳生成8位不重复随机码

时间戳是自 1970 年 1 月 1 日（00:00:00 GMT）至当前时间的总秒数。由于时间都不会重复，所以利用时间来生成一串不重复的ID或字符串就非常方便。 思路：获取当前时间的时间戳，然后转换为十六进制。

生成结果如下： 当前时间：Mon May 13 14:47:51 CST 2013生成8位随机码：9ca52f20

相关代码： import java.util.Date;

   public static void main(String[] args ) {
         // TODO Auto-generated method stub
    System.out.println(new Date());
    System.out.println(toHex(new Date().getTime()));

  }

/**
 * 获取8位不重复随机码（取当前时间戳转化为十六进制）
 * @author ShelWee
 * @param time
 * @return
 */
   public static String toHex(long time){    
         return Integer.toHexString((int)time);
  }
