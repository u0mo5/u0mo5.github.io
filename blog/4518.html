mysql给root开启远程访问权限，修改root密码

Tue 09 May 2017
By u0mo5

1.MySql-Server 出于安全方面考虑只允许本机(localhost, 127.0.0.1)来连接访问. 这对于 Web-Server 与 MySql-Server 都在同一台服务器上的网站架构来说是没有问题的. 但随着网站流量的增加, 后期服务器架构可能会将 Web-Server 与 MySql-Server 分别放在独立的服务器上, 以便得到更大性能的提升, 此时 MySql-Server 就要修改成允许 Web-Server 进行远程连接.2.不用每次都登到服务器去添加修改表，只要用图形化界面即可远程管理。 我们可以按照下面的步骤修改: 1, 登录 Mysql-Server 连接本地 mysql (默认只允许本地连接)  2, 修改 Mysql-Server 用户配置mysql> USE mysql; --> SELECT User, Password, Host FROM user; --+------+----------+-----------+

| User | Password | Host      | +------+----------+-----------+ | root |          | localhost | +------+----------+-----------+ row   (> --> -- 下面我们另外添加一个新的 root 用户, 密码为空, 只允许 .> GRANT ALL PRIVILEGES ON . TO @ IDENTIFIED BY > -- @可以替换为@‘%> -- UPDATE user SET Host= WHERE User= AND Host= LIMIT ;
